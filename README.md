# Online Food Ordering and Delivery System

This is a web-based food ordering and delivery application developed using Django.

## Features
- User Registration and Login
- Browse Food Menu
- Add to Cart
- Place Orders
- Admin Dashboard
- Food Image Upload

## Technologies Used
- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite / MySQL

## Installation

1. Clone the repository:
git clone https://github.com/yourusername/fooddelivery.git

2. Go to project folder:
cd fooddelivery

3. Install dependencies:
pip install django

4. Run migrations:
python manage.py makemigrations
python manage.py migrate

5. Run server:
python manage.py runserver

## Output
Open browser and visit:
http://127.0.0.1:8000/
